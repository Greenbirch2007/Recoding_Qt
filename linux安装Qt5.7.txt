在linux上安装Qt5.7
1.准备
在安装Qt5.7之前我们需要先安装gcc和g++，确保电脑上有c语言和c++的编译环境。

apt-get install gcc g++

除此之外我们还需要准备调试引擎用于qt调试使用。这里我们安装gdb就可以了。

apt-get install gdb

2.下载Qt安装程序

3.安装
下载完安装文件以后就需要在电脑上安装了。在linux上安装的时候需要权限问题。

chmod a+x qt-opensource-linux-x64-5.7.0.run
安装的过程当中我们需要登陆或者注册，有账号就直接登陆，没账号在下面的sign-up注册就可以了。
然后一路下一步就安装好了。

4.注意
安装好以后并不会主动创建桌面图标，所以这时候建议安装好后启动程序，然后讲图标固定或者驻留在docker上。
